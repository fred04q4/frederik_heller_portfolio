Grundlæggende Animation

lærgning om CSS animation og JavaScript gmmen spil udvikningen

I tema - 4 Blve vi introduceret til JavaScript igemmen at lave et spil og hvad JavaScript kan og bruges til i pogramerning.

JavaScript bruges blandt andet til at udføre bruger intaktioner på fx din hjemmeside, det er det der får din fx bruger-menu eller info box til at virke når man trykke på den.

Det gør den ved at du op sætter et par stykker CSS koder op, som du kan gå ind i JavaScript og sætte en funktion op hvor du ænden skift i mellem med toggle,

eller lave en if og is som går ind og kigger på hvordan en spesefik ting er og vis det opfylder dit kreter så gør din JavaScript så det du har bedt den om.

Vi fik også insigt i adobe Illustrerer og hvordan man designer ting i det, samt mere desing mæsige værktøjer til tegne tekniker og ide-udviklning på historie og spil elementer.
